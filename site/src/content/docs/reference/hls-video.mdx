HLS video element that uses [hls.js](https://github.com/video-dev/hls.js/) for adaptive bitrate streaming. This is the recommended choice for HLS playback with full feature support across all browsers. For alternatives, see [NativeHlsVideo](/docs/framework/html/reference/native-hls-video/) (browser-native HLS) or [SimpleHlsVideo](/docs/framework/html/reference/simple-hls-video/) (lightweight HLS).
